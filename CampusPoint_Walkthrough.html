Apa itu CampusPoint?

Sistem manajemen poin dan sertifikat berbasis blockchain untuk kampus. Mahasiswa mendapatkan token CPNT (CampusPoint) dan sertifikat NFT saat mengikuti kegiatan kampus.

// State Variables
address public owner                    // Alamat pemilik kontrak
ICampusPoint public campusPoint        // Referensi ke kontrak token
IActivityCertificate public activityCert // Referensi ke kontrak NFT
uint256 public nextActivityId          // ID kegiatan berikutnya
mapping(uint256 => Activity) activities // Mapping kegiatan

// Events
event ActivityCreated(uint256 id, string name, uint256 pointReward)
event StudentRewarded(uint256 activityId, address student, uint256 pointReward)
event CertificateMinted(uint256 activityId, address student, uint256 tokenId, string uri)
